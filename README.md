## Projects

### main.py — AI Chatbot
A simple chatbot that responds to basic questions.
- Concepts: while loops, if/elif/else, string methods

### basics_practice.py — Python Fundamentals
Core Python basics practiced from scratch.
- Concepts: variables, loops, lists, functions, dictionaries

### function_exercises.py — Function Practice
5 exercises covering all types of Python functions.
- Concepts: parameters, return values, connecting functions

### diary_app.py — Diary Application
Save and read personal diary entries permanently.
- Concepts: file handling, error handling, menus, loops

### restaurant_ordering.py — Restaurant System
Full ordering system with menu, bill and file logging.
- Concepts: nested dictionaries, functions, datetime, random

### student_class.py — Student Class
Student blueprint with automatic grade calculation.
- Concepts: classes, __init__, methods, self keyword

### bank_account.py — Bank Account Class
Bank account with deposit and withdraw functionality.
- Concepts: OOP, data changing over time, conditionals

### contact_book.py — Contact Book
Save and search contacts by name.
- Concepts: classes, list of dictionaries, search logic
